# Assets Folder

This folder contains all app assets including images, icons, fonts, and other resources.

## Directory Structure

```
assets/
├── fonts/          # Custom font files (.ttf, .otf)
├── icons/          # App icons and custom icon images
├── images/         # Product images, banners, categories, etc.
└── README.md       # This file
```

## Required Assets

### App Icons (Place in root or icons/)
- `icon.png` - App icon (1024x1024) for iOS
- `adaptive-icon.png` - Android adaptive icon (1024x1024)
- `favicon.png` - Web favicon (48x48)

### Splash Screen (Place in root or images/)
- `splash.png` - Splash screen image (1242x2436 for iOS, 1080x1920 for Android)

## Asset Sizes

For best results, use the following sizes:

- **App Icon**: 1024x1024 pixels (PNG)
- **Adaptive Icon**: 1024x1024 pixels (PNG)
- **Splash Screen**: 
  - iOS: 1242x2436 pixels (PNG)
  - Android: 1080x1920 pixels (PNG)
- **Favicon**: 48x48 pixels (PNG)
- **Product Images**: 800x800 pixels (JPG/PNG)
- **Banner Images**: 1080x1920 pixels (JPG/PNG)

## Subdirectories

### fonts/
Contains custom font files (.ttf, .otf). See `fonts/README.md` for details.

### icons/
Contains custom icon images and app icons. See `icons/README.md` for details.

### images/
Contains all app images including products, banners, categories. See `images/README.md` for details.

## Creating Assets

You can create these assets using design tools like:
- Figma
- Adobe Photoshop
- Canva
- Sketch

Or use online tools like:
- [AppIcon.co](https://www.appicon.co/)
- [MakeAppIcon](https://makeappicon.com/)
- [IconKitchen](https://icon.kitchen/)

## Current Implementation

For development, the app uses placeholder images from placeholder services. In production, replace these with actual assets stored in the appropriate subdirectories.

## Notes

- `android/` and `ios/` directories are auto-generated by Expo when you run `expo prebuild` or build the app
- Assets in this folder are automatically bundled with your app
- Keep file sizes optimized for better app performance
